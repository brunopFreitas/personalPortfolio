Bruno Freitas Portfolio 2022

Flight Tracker

Tracking Canadians Airplanes

This was a very cool college project. Basically, I have used Leaflet.js with a GeoJSON object and flight data retrieved from the opensky-network. The main goal was to track Canadian Airplanes. The map is interactive and every plane has its own information, such as ID, velocity, Origin Country, and if it has landed.

Truco Card Game

Truco is a very popular Brazilian card game, this game was developed as a final project for my JavaScript class in college.

Try to beat the PC in this cool Brazilian card game. Developed using HTML5, CSS3, JS, Saas and Animaleto.js. Cards are retrieved from deckofcardsapi.
