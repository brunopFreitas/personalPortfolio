About

What I have learned so far

  • Front-End

    HTML5, CSS3, Sass, JavaScript and Bootstrap

    Creating websites using these technologies was the main part of my program.

  • Back-End

    Java, Node, PHP

    The main back-end language that I know is Java. It was with Java that I was taught all the OOP concepts. I know a bit about Spring framework and how to handle databases communication using hibernate. In my internship I have developed a module in Node to help parse a CSV file to a Json. I'm currently learning PHP and Laravel.

  • Database

    Relational Databases

    MySQL and Postgres was the main databases I have worked so far. It was working with these that I have learned how to DDL and DML using SQL statements.

  • Software Integration

    Rest, Soap, Pentaho

    I have worked a lot with software integration as a software analyst. I have helped developing the architecture, json and xml objects, and testing. I have used pentaho a lot as well, both as an integration tool and BI report generator.
